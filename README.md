## Modrinth auto-updater tutorial:
- 1. Import the .mcpack file.
- 2. Once imported, open the **instance specific** settings.
- 3. Navigate to the `` section of the settings pop-up.
- 4. Check the `custom` box.
- 5. In the `pre-launch` text box, enter:
`java -jar CreateVC-Updater.jar https://jammersmurph.github.io/CreateVC/pack.toml`
- 6. Close the settings window.
- 7. Launch your game.
- 8. Congratulations! your modpack will now update automatically feom now on!
